# buildwithnexus-linux-arm64

Prebuilt `buildwithnexus` binary for **linux arm64** (`aarch64-unknown-linux-gnu`).

This package is an internal platform artifact of [buildwithnexus](https://www.npmjs.com/package/buildwithnexus) — install that package instead; npm selects the right platform package automatically via `optionalDependencies`.

The binary is built on GitHub Actions from the tagged release commit, SHA-256-verified when packaged, and carries a build-provenance attestation.
